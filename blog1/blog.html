Getting started with internal attachment

Introduction

Internal practical attachment basically is aimed at according students opportunity
to interact with the physical concepts of their course work. In JKUAT, the practice
usually embarks after the second academic year for a period of about 8weeks of
intense practice and work.

Skills

Ever since i started my course work, it has been hands-on for me regarding
the skillful side of things. I started of with C language which really brought
to my attention the basic syntax of coding before moving to Java and Python
during the second semester of my first year.

I have to admit, being a freshman, handling both Java and Python in one semester
was not just a walk in the park. (clearly those who have tried to do a bit of java can
agree that its learning curve grows exponentially as you progress unlike other
programming languages like Python with little to no boiler plate code).

Later on i had to learn some C++ which to me wasn't a problem as its syntax
compared to java is quite similar except for some changes in function calling
and object-function manipilation.
